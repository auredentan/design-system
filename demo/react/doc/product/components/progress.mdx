```javascript import
import { Progress, ProgressVariant } from 'LumX';
```

# Progress

**Progress notify user that an action is being completed.**

Progress come in two styles: _circular_ and _linear_.

## Circular

```javascript jsx disableGrid
() => (
    <>
        <Progress />
    </>
);
```

## Linear

```javascript jsx disableGrid
() => (
    <>
        <Progress variant={ProgressVariant.linear} />
    </>
);
```
