# Tooltip

**Tooltips display informations when users hover over or focus on an element.**

## Placement

Tooltips may be positioned _top_, _bottom_, _left_, or _right_ to the trigger element.

```javascript jsx withThemeSwitcher
() => (
    <>
        <img width="420" src="demo/react/doc/product/components/tooltip-postion.png" />
    </>
);
```

There are two variants of tooltips: _default_ and _rich_.
