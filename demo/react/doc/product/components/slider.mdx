# Slider

## Default
```javascript jsx withThemeSwitcher disableGrid
(theme) => {
    const [value, setValue] = useState(0);
    return (
        <>
            <pre>{`Adjust threshold: ${value}`}</pre>
            <br />
            <Slider
                label='Default'
                helper='This is an helper text'
                max={10}
                min={0}
                theme={theme}
                defaultValue={4}
                onChange={val => {
                    setValue(val);
                }}
            />
        </>
    )
};
```
