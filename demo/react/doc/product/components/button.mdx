# Button

**Buttons trigger actions with a single click.**

## High emphasis

Use high emphasis to help users to identify the main action.
There should not be more than one high emphasis button in a view.

```javascript jsx withThemeSwitcher
(theme) => (
    <>
        <Button theme={theme}>Default</Button>
        <Button disabled={true} theme={theme}>
            Disabled
        </Button>
        <Button leftIcon={mdiPencil} theme={theme}>
            With Icon
        </Button>
        <DropdownButton
            dropdown={
                <List isClickable>
                    <ListItem size={ListItemSize.tiny}>Los Angeles</ListItem>
                    <ListItem size={ListItemSize.tiny}>Monterrey</ListItem>
                    <ListItem size={ListItemSize.tiny}>Georgetown</ListItem>
                    <ListItem size={ListItemSize.tiny}>Cali</ListItem>
                    <ListItem size={ListItemSize.tiny}>Trondheim</ListItem>
                </List>
            }
            theme={theme}
        >
            Dropdown
        </DropdownButton>
        <DropdownButton
            dropdown={
                <List isClickable>
                    <ListItem size={ListItemSize.tiny}>Los Angeles</ListItem>
                    <ListItem size={ListItemSize.tiny}>Monterrey</ListItem>
                    <ListItem size={ListItemSize.tiny}>Georgetown</ListItem>
                    <ListItem size={ListItemSize.tiny}>Cali</ListItem>
                    <ListItem size={ListItemSize.tiny}>Trondheim</ListItem>
                </List>
            }
            splitted={true}
            theme={theme}
        >
            Split
        </DropdownButton>
        <IconButton icon={mdiPlus} theme={theme} />
    </>
);
```
