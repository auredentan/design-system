```javascript import
import { RadioButton, RadioGroup, RadioButtonHelper, RadioButtonLabel, Theme } from 'LumX';
```

# Radio button

**Radio buttons display exclusive options to chose from.**

## Default

```javascript jsx withThemeSwitcher disableGrid
(theme) => (
    <>
        <RadioGroup>
            <RadioButton theme={theme} value="lorem" name="test1" checked={true}>
                <RadioButtonLabel>Radio button 1</RadioButtonLabel>
            </RadioButton>
            <RadioButton disabled theme={theme} value="ipsum" name="test1" checked={false}>
                <RadioButtonLabel>Radio button 2</RadioButtonLabel>
            </RadioButton>
            <RadioButton theme={theme} value="dolor" name="test1" checked={false}>
                <RadioButtonLabel>Radio button 3</RadioButtonLabel>
            </RadioButton>
        </RadioGroup>
    </>
);
```

## With optional helper text

```javascript jsx withThemeSwitcher disableGrid
(theme) => (

            <RadioButton theme={theme} value="lorem" name="test1" checked={true}>
                <RadioButtonLabel>Radio button with helper 1</RadioButtonLabel>
                <RadioButtonHelper>
                    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed posuere faucibus efficitur.{' '}
                </RadioButtonHelper>
```
