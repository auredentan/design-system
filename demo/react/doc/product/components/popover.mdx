# Popover

**Popovers display pieces of UI on top of others. They are positioned from anchor elements.**

## Default

```javascript jsx withThemeSwitcher
(theme) => {
    const demoPopperStyle = {
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        width: 200,
        height: 100,
    };

    const demoPopoverHolderStyle = {
        alignItems: 'start',
        display: 'flex',
        height: 132,
        justifyContent: 'center',
    };

    const anchorRef = React.useRef(null);
    const popoverRef = React.useRef(null);

    const { computedPosition, isVisible } = Popover.useComputePosition(Placement.BOTTOM, anchorRef, popoverRef, true);

    return (
        <>
            <div style={demoPopoverHolderStyle}>
                <Chip chipRef={anchorRef} theme={theme} size={Size.s}>
                    Anchor
                </Chip>
            </div>
            <Popover theme={theme} popoverRect={computedPosition} popoverRef={popoverRef} isVisible={isVisible}>
                <div style={demoPopperStyle}>{'Popover'}</div>
            </Popover>
        </>
    );
};
```

## Positions

### Directed

Popovers can be placed to 4 positions :
_TOP_, _RIGHT_, _BOTTOM_, _LEFT_

### Auto

Popovers can also be placed automatically where space is available. They will be placed at the bottom of anchors by default.

```javascript jsx withThemeSwitcher
(theme) => {
    const demoPopperStyle = {
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        width: 200,
        height: 100,
    };

    const demoPopoverHolderStyle = {
        alignItems: 'center',
        display: 'flex',
        height: 224,
        justifyContent: 'center',
    };

    const anchorRef = React.useRef(null);
    const popoverRef = React.useRef(null);

    const { computedPosition, isVisible } = Popover.useComputePosition(Placement.AUTO, anchorRef, popoverRef, true);

    return (
        <>
            <div style={demoPopoverHolderStyle}>
                <Chip chipRef={anchorRef} theme={theme} size={Size.s}>
                    Anchor
                </Chip>
            </div>
            <Popover theme={theme} popoverRect={computedPosition} popoverRef={popoverRef} isVisible={isVisible}>
                <div style={demoPopperStyle}>{'Popover'}</div>
            </Popover>
        </>
    );
};
```

## Offset

Popovers positions can have an offset.

```javascript jsx withThemeSwitcher
(theme) => {
    const demoPopperStyle = {
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        width: 200,
        height: 100,
    };

    const demoPopoverHolderStyle = {
        alignItems: 'start',
        display: 'flex',
        height: 132,
        justifyContent: 'center',
    };

    const anchorRef = React.useRef(null);
    const popoverRef = React.useRef(null);

    const { computedPosition, isVisible } = Popover.useComputePosition(Placement.RIGHT, anchorRef, popoverRef, true, {
        horizontal: 30,
        vertical: 60,
    });

    return (
        <>
            <div style={demoPopoverHolderStyle}>
                <Chip chipRef={anchorRef} theme={theme} size={Size.s}>
                    Anchor
                </Chip>
            </div>
            <Popover theme={theme} popoverRect={computedPosition} popoverRef={popoverRef} isVisible={isVisible}>
                <div style={demoPopperStyle}>{'Popover'}</div>
            </Popover>
        </>
    );
};
```

## Alignements

In addition to positions, popovers can be aligned to a chosen anchor's side :
_TOP_START_, _TOP_END_, _RIGHT_START_, _RIGHT_END_, _BOTTOM_START_, _BOTTOM_END_, _LEFT_START_, _LEFT_END_, _AUTO_START_, _AUTO_END_,
