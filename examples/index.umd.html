This is an example of using LumX in UMD

All you have to do is import angularjs.lumx.css in your page's head and angularjs.lumx.js at the end of your page's body after AngularJS library.

You can then initialize an application using any LumX available Dependency Injection and use any LumX directive in your templates. You can also use any CSS classes of LumX in your templates.

Here is a basic example

                <!DOCTYPE html>
                <html ng-app="lumx-example-app">
                    <head>
                        <title>Basic example</title>
                        <link rel="stylesheet" href="node_modules/@lumx/dist/angularjs.lumx.css" />
                    </head>

                    <body class="p++">
                        <div class="lx-typography-title">This is a basic example</div>

                        <div class="mt++"><lx-button>Nice!</lx-button></div>

                        <script
                            src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.3.1/jquery.min.js"
                            type="text/javascript"
                        ></script>
                        <script
                            src="https://cdnjs.cloudflare.com/ajax/libs/angular.js/1.7.7/angular.min.js"
                            type="text/javascript"
                        ></script>
                        <script src="node_modules/@lumx/dist/angularjs.lumx.js" type="text/javascript"></script>

                        <script type="text/javascript">
                            angular
                                .module('lumx-example-app', ['@lumx/angularjs'])
                                .run(function(LxNotificationService) {
                                    LxNotificationService.success('Basic example application successfully started!');
                                });
                        </script>
                    </body>
                </html>
OK, got it!
